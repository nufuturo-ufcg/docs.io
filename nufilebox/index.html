This API provides a solution for transferring files over the network. This API was build using clojure Pedestal API, which is a a set of libraries that we use to build services and applications. Some useful links:

init-multipart

Returns a set of pre-signed upload urls for each part of the file to be uploaded. User needs to be autenticated by Okta to use this endpoint

/init-multipart

header Parameters
Authorization
required
string <JWT>

The JWT token used to authenticate the user

Request Body schema: application/json
required

Body of the init-multipart request

filename
string

Name of the file

password
string

Password used to encrypt the file

ttl
int

Number of days for the file to expire

max-downloads
int

Number of downloads available for the file

parts-number
int

Number of parts that the file is going to be send at. Also means the number of pre-signed urls that will be returned

Responses

http://file-transfer-ufcg.com/api/init-multipart

complete-multipart

Completes the multipart upload process started for the file to be uploaded. User needs to be autenticated by Okta to use this endpoint

/complete-multipart

header Parameters
Authorization
required
string <JWT>

The JWT token used to authenticate the user

Request Body schema: application/json
required

Body of the complete-multipart request

file-id
string

UUID of the uploaded file

etags
list

A list of etags for all uploaded parts of the file

checksum-parts
list

A list of the checksums for all the chunks of the file

Responses

http://file-transfer-ufcg.com/api/complete-multipart
